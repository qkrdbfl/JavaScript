브라우저 환경의 전역 객체 window의 대표 메소드

open

window.open(url, name, params) 메서드로 새 창을 열 수 있다.
url : 새 창에 로드할 url, name : 새 창의 이름으로 해당 이름을 가진 창이 이미 있으면 그 안에서 열리고 그렇지 않으면 새창이 열림, params : 새 창의 설정을 쉼표로 구분하여 문자열로 전달

alert

window.alert(message) 메서드는 확인 버튼을 가지며 메세지를 지정할 수 있는 경고 대화상자를 띄운다.

confirm

window.confirm(message) 메서드는 확인과 취소 두 버튼을 가지며 메세지를 지정할 수 있는 모달 대화상자를 띄운다. 반환 값은 확인을 누를 시 true, 취소를 누르거나 ESC 키를 누르면 false이다.

prompt

window.prompt(message, default) 메서드는 사용자가 텍스트를 입력할 수 있도록 안내하는 선택적 메세지를 갖고 있는 대화 상자를 띄운다. default는 텍스트 입력 필드에 기본으로 채워 넣을 문자열이다. 반환 값은 확인을 누를 시 사용자가 입력한 문자열이며 취소를 누르거나 ESC 키를 누르면 null이다.
